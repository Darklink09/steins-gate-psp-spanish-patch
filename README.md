# Steins;Gate (PSP) — Parche en español

Parche de traducción al español para **Steins;Gate de PSP**.

## Autores

- **Traducción, adaptación, inserción y port:** Darklink09
- **Testeo y apoyo adicional:** Vampiro49

## Propósito

Este parche fue hecho con fines de preservación y accesibilidad para la comunidad.

## Créditos

- Parche original en inglés para PSP de **yar-sh**:  
  https://github.com/yar-sh/steins-gate-psp-patch

- Recursos visuales de apoyo, incluyendo ciertos videos e imágenes, tomados o adaptados a partir del proyecto de traducción de **Tradusquare**:  
  https://tradusquare.es/proyectos/steinsgate/

## Versiones

### v1.0.1

Corrección menor del parche.

Cambios:

- Correcciones pequeñas en signos de puntuación.
- Corrección de un cierre inesperado en el capítulo 4, durante la escena en la casa de Faris.
- Arreglo de un bug de sonido en la introducción del juego, donde el audio se escuchaba demasiado bajo.

### v1.0

Primera versión pública del parche.

Incluye:

- Historia traducida al español.
- DICT/tips traducidos.
- Mensajes del teléfono traducidos.
- Fondos, videos y recursos visuales traducidos o adaptados.
